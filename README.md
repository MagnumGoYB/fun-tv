# fun-tv

> 一个有趣的 Top20 电影榜单

- [Website](https://magnumgoyb.github.io/fun-tv/)

## 本地开发

克隆项目到本地：

```bash
$ git clone git@github.com:MagnumGoYB/fun-tv.git
$ cd fun-tv
$ npm install
$ npm start
```

构建：

```bash
$ npm run build
```
